Nuestros Programas

Programador Front-end

En este curso de Programador Front End aprenderás a crear un sitio web, y te sumergirás en las mejores prácticas del diseño web, trabajando con HTML y CSS. Además, sabrás cómo programar con Javascript y React Js.

Programador Back-end

En este curso aprenderás a desarrollar aplicaciones modernas con Node.js y MongoDB. Programarás en Javascript del lado del servidor aplicando técnicas asincrónicas. Aprenderás a trabajar con Bases de datos SQL y NoSQL.

Programador Python

En este curso aprenderás las bases de programación. Incorporarás los conocimientos necesarios de Class-Based-View, login - register - accounts, CRUD y modelforms. Desarrollarás una aplicación web estilo blog de la mano de Python en Django Framework.
